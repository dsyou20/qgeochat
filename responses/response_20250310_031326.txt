QGIS에서 레이어 정보를 출력하는 코드를 작성하려면, QGIS Python API를 사용하여 활성화된 레이어의 정보를 가져와 출력할 수 있습니다. 아래는 활성화된 레이어의 기본 정보를 출력하는 예제 코드입니다.

```python
from qgis.core import *
from qgis.utils import iface

def print_layer_info():
    """활성화된 레이어의 정보를 출력합니다."""
    try:
        # 활성화된 레이어 가져오기
        layer = iface.activeLayer()
        if not layer:
            print("활성화된 레이어가 없습니다.")
            return
        
        # 레이어 정보 출력
        print(f"레이어 이름: {layer.name()}")
        print(f"레이어 타입: {layer.type()}")
        print(f"피처 수: {layer.featureCount()}")
        print(f"좌표계: {layer.crs().authid()}")
        
        # 벡터 레이어인 경우 필드 정보 출력
        if layer.type() == QgsMapLayer.VectorLayer:
            fields = layer.fields()
            for field in fields:
                print(f"필드 이름: {field.name()}, 유형: {field.typeName()}")
        
        print("레이어 정보 출력 완료!")
        
    except Exception as e:
        print(f"오류 발생: {str(e)}")

if __name__ == '__main__':
    print_layer_info()
```

이 코드는 QGIS의 Python 콘솔에서 실행할 수 있으며, 활성화된 레이어의 이름, 타입, 피처 수, 좌표계, 그리고 벡터 레이어의 경우 필드 정보를 출력합니다.
